# Spam Detection using Machine Learning

A Python-based spam email classifier using NLP and Machine Learning techniques.

## What it does
- Classifies emails/SMS as Spam or Ham
- Uses TF-IDF for feature extraction
- Compares Naive Bayes and SVM models
- Achieves ~98% accuracy

## Tools Used
Python | scikit-learn | NLTK | Pandas | Matplotlib | Seaborn

## Models
- Naive Bayes: ~97% accuracy
- SVM: ~98% accuracy

## Dataset
SMS Spam Collection Dataset (Kaggle)
